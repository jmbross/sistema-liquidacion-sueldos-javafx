# ADR 0001: Rebuild incompatible academic iterations

- Status: Accepted
- Date: 2026-07-21

## Context

Three related repositories contained overlapping names but incompatible packages, schemas, incomplete features, embedded binaries, and unsafe data practices.

## Decision

Keep the designated canonical lineage as the case-study origin while rebuilding the publishable tree. Rescue verified concepts—not files—such as repository separation, roles, worker assignment, `BigDecimal`, and dashboard navigation.

## Consequences

The result is coherent and testable but is not a line-by-line preservation of every academic prototype. Historical repositories remain private, sanitized, and archived as prior iterations.
