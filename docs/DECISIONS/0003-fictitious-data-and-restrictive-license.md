# ADR 0003: Fictitious data and restrictive license

- Status: Accepted
- Date: 2026-07-21

## Decision

Version only clearly fictitious `DEMO-` records and `.local` email addresses. Publish the source for portfolio evaluation under an all-rights-reserved notice rather than assume that all historical third-party rights permit a permissive license.

## Consequences

The repository is reviewable but reuse requires permission. This conservative choice can be revisited after a provenance review.
