# Changelog

## [1.0.0-portfolio] - 2026-07-21

### Added
- Java 21 and JavaFX 21 Maven build with verified wrapper.
- Layered payroll domain, JDBC repositories, application services, and JavaFX UI.
- Flyway schema, fictitious BCrypt demo accounts, and Docker Compose MySQL 8.4.
- Role-aware CRUD, worker assignment, payroll calculation, persistence, and PDF export.
- Unit/integration tests, JaCoCo, Spotless, SpotBugs, CodeQL, Dependabot, and secret scanning.
- Architecture, development, threat-model, AI-engineering, and decision documentation.

### Security
- Removed unsafe historical data and artifacts from the published history.
- Replaced plaintext SQL authentication and hardcoded credentials.

### Limitations
- Portfolio case study only; no native installer or production payroll rules.
